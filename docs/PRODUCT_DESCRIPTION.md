## 产品简介（精简版）

Task Breakdown Tool 是一款 AI 驱动的任务拆解工具。只需输入你的目标，系统会智能生成从月度到每日的可执行计划。通过甘特图可视化、每日任务优先级推荐和复盘功能，帮助你专注执行，让大目标变成每天的小进步。

---

## 核心功能描述

### 1. 对话式目标拆解
如同展开一张古老的羊皮纸卷轴，通过九种不同类型的问题，系统会温柔地引导你思考目标的各个方面——从截止日期到经验水平，从可用时间到潜在阻碍。AI 会根据你的回答，智能生成个性化的任务拆解方案。

### 2. 三层计划体系
系统自动生成**月度目标 → 周里程碑 → 每日任务**的三层结构，让长期规划与短期执行完美衔接。每日任务自动分配到四象限，为你推荐 Top 1/2/3 优先任务，让你每天都知道该先做什么。

### 3. 甘特图可视化
一眼看穿整个任务的时间线和依赖关系。里程碑节点清晰标注，进度实时更新，让抽象的计划变得直观可见。

### 4. 每日复盘与优化
每天结束时花几分钟复盘，记录完成情况与未完成原因。系统会根据你的反馈自动调整后续任务，让计划越来越贴合实际。

### 5. 迭代优化
对 AI 生成的任务不满意？可以通过补充问题提供更多信息，让 AI 重新生成更精准的计划。
